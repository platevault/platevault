# Bundled fonts

Inter v4.1 (2024-11-15), hinted woff2 subset, from the official release:

- Release: https://github.com/rsms/inter/releases/download/v4.1/Inter-4.1.zip
- Source path inside the zip: `extras/woff-hinted/`
- License: SIL Open Font License 1.1 (`LICENSE.txt`, unmodified from the release)

Only the six weights/styles used by the UI (400/500/600, normal/italic) are
bundled. `InterVariable` and `InterDisplay` files are intentionally excluded.
