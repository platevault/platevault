# Implementation Plan: IPC wrapper removal (adopt generated bindings)

**Branch**: `037-ipc-wrapper-removal` | **Spec**: [spec.md](./spec.md) | **Research**: [research.md](./research.md)

## Summary

Make the generated tauri-specta bindings the single source of truth for every IPC call. Route
their dispatch through the existing mock/recorder switcher (extracted to `api/ipc.ts`), unwrap
the `Result` into the throw-on-error contract, then migrate callers off the hand-written
`commands.ts` wrappers feature-by-feature and delete the wrapper layer. Eliminates the
mock-hidden, Windows-only IPC drift bug class.

## Technical Context

- **Frontend**: React 19 + TS, Vite, vitest; `apps/desktop/src`.
- **Bindings**: `apps/desktop/src/bindings/index.ts`, generated by `apps/desktop/src-tauri/tests/bindings.rs`
  via `Builder::export(Typescript::default())` (tauri-specta 2.0.0-rc.25 / specta-typescript 0.0.12).
- **Dispatch today**: `commands.ts` `invoke()` switcher (mock / `_invokeOverride` / real).
- **Mocks**: `apps/desktop/src/api/mocks.ts` `mockInvoke(cmd, args)`, keyed by snake_case command name.
- **Callers**: 56 files import `@/api/commands`. **Commands**: ~90 wrappers / 86 names.
- **Dev recorder**: spec-021, `dev-tools` cargo feature, installs `_invokeOverride`.

## Phases

- **Phase 1 — Foundation (US2)**: `api/ipc.ts` switcher + `unwrap`; redirect invoke import in
  `bindings.rs`; regenerate bindings; prove mock mode + recorder still work. No caller changes.
- **Phase 2 — Transitional delegation (US3)**: rewrite `commands.ts` wrappers as
  `unwrap(commands.*)` (removes invoke literals from `commands.ts`); guard test still green or
  updated. Callers/tests untouched.
- **Phase 3 — Caller migration (US1)**: per feature area, repoint imports `@/api/commands` →
  `@/bindings` (`commands.*` + `unwrap`); migrate that area's test mocks to the dispatch layer.
- **Phase 4 — Teardown**: delete `commands.ts` + `commands.bindings-guard.test.ts`; assert no
  `@/api/commands` imports and no `invoke('...')` literals outside `api/ipc.ts` (SC-001/SC-005).
