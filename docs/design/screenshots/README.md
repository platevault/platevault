# App screenshots

Full-window captures of every page of the Astro Library Manager desktop app,
taken from the real Windows build (Tauri dev, `desktop_shell.exe`) via the MCP
bridge at 2660×1194. Captured against an empty test library (most pages show
their empty state); the Archive page carries sample seed data.
